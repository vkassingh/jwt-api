<!-- Run Locally -->
## :running: Run Locally

Clone the project

```bash
  git clone https://github.com/vkassingh/jwt-api.git
```

Go to the project directory

```bash
  cd clever-crypto
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
 nodemon
```
